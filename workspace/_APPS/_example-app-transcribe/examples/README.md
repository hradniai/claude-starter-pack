# Examples

Drop input audio/video files here for testing.

Recommended test files:
- A 1-minute clear-speech sample (validates transcription accuracy)
- A multi-speaker sample (validates diarization, if implemented)
- A noisy or accented sample (validates edge cases)

Outputs (`*.transcript.md`) will be written next to the input files.
