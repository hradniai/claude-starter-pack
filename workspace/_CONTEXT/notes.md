# Notes

Brain dump for ideas, impulses, and things to explore. NOT a task list.

Format: each entry starts with a `## YYYY-MM-DD HH:MM` timestamp.

Add `→ research` to any entry to dispatch auto-research via the notes-research hook (requires `ANTHROPIC_API_KEY` in `~/.claude/.env`).

---

## 2026-01-01 09:00

Example note. Replace this with your own first entry.

## 2026-01-01 09:15

Example note with research marker. → research

This entry would trigger the notes-research hook to dispatch research and write the result to `research/example-note-research-2026-01-01.md`.
